Payment plugin for woocommerce using TPro2 by 2C Processor USA, LLC ( 2CP ).

== Description ==

This plugin is an addon for WooCommerce to implement a payment gateway method for accepting Credit Cards Payments 
by merchants using 2C Processor's Transaction Pro (TPro2) payment gateway. 

Plugin uses clientid, merchantaccountid, and token to access TPro gateway.

== Installation ==

1. upload the plugin using the Add New menu item in the menubar.
